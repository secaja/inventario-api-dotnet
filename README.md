# Inventario API

## Tecnologías
- .NET 9
- PostgreSQL
- Entity Framework Core
- JWT

## Cómo ejecutar
1. Configurar conexión en appsettings.json
2. Ejecutar migraciones
3. dotnet run

## Endpoints
- POST /auth/login
- GET /productos/inventario
- POST /productos/movimiento
